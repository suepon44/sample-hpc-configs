### ライセンス利用状況確認方法

```bash
# FlexLM: ライセンス利用状況の確認
lmstat -a -c （ライセンスファイルパス）

# FlexLM: 特定フィーチャーの利用状況確認
lmstat -f （フィーチャー名） -c （ライセンスファイルパス）

# RLM: ライセンス利用状況の確認
rlmstat -a -c （ポート番号）@（サーバ名）

# RLM: 特定ISVの利用状況確認
rlmstat -i （ISV名） -c （ポート番号）@（サーバ名）
```
