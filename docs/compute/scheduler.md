## Prolog/Epilog実行フロー

```mermaid
sequenceDiagram
    participant User as ユーザー
    participant Sched as スケジューラ
    participant Prolog as Prologスクリプト
    participant Node as 計算ノード
    participant Epilog as Epilogスクリプト

    User->>Sched: ジョブ投入
    Sched->>Sched: 優先度計算・スケジューリング
    Sched->>Prolog: Prolog実行
    Prolog->>Node: 環境準備
    Node->>Node: ジョブ実行
    Node->>Epilog: ジョブ終了通知
    Epilog->>Node: 環境クリーンアップ
    Epilog->>Sched: 完了報告
```
