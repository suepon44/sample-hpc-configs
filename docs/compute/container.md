## コンテナ利用フロー

```mermaid
flowchart TD
    A[ユーザー: イメージ準備] --> B{イメージの取得元}
    B -->|公開レジストリ| C[Docker Hub等からpull]
    B -->|プライベートレジストリ| D[社内レジストリからpull]
    B -->|自作| E[Dockerfileからbuild]
    C --> F[イメージをプライベートレジストリに登録]
    D --> G[ジョブスクリプトにコンテナ指定]
    E --> F
    F --> G
    G --> H[ジョブ投入]
    H --> I[スケジューラがコンテナ起動]
    I --> J[コンテナ内でジョブ実行]
    J --> K[結果出力・コンテナ終了]
```
